# Punjabi Display Font

To enable `raaj___5` in the homepage hero, add a font file you are licensed to use at one of these paths:

- `public/fonts/raaj___5.woff2` (preferred for production)
- `public/fonts/raaj___5.ttf` (fallback)

The hero uses a system fallback until one of these files is present. Do not hotlink the font from a third-party generator site; self-host only a font file whose licence permits web distribution.
