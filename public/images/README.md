# Homepage Image Slots

Drop images here to activate backgrounds and thumbnails. Components fall back to gradients when files are missing.

| File | Used in | Recommended size |
|------|---------|-------------------|
| `hero-bg.jpg` | Hero section full background | 1920×1080px min |
| `hero-thumb.jpg` | Currently Playing card thumbnail | 200×200px |
| `continue-journey.jpg` | Continue Your Journey card | 800×450px |
| `temple-outline.png` | Daily Wisdom decorative background | 400×300px transparent PNG |
| `gurbani-cta.jpg` | CTA banner (Audio by Themes) | 600×400px |

**Hero image tip:** Show speaker/turban on right, Golden Temple sunset works well. Left side stays dark for text readability.

**Component comments:** Each component also marks `IMAGE SLOT` / `BACKGROUND IMAGE SLOT` in source for dynamic CMS wiring later.
